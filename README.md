# Carrot Market
Next.js로 제작한 '당근마켓' 앱 풀스택 클론코딩 (프론트엔드, 백엔드) <br />
<br />
<기술스택><br />
✓ React 18 <br />
✓ Next.JS <br />
✓ Typescript <br />
✓ TailwindCSS <br />
✓ SWR <br />
✓ Prisma <br />
✓ Planet Scale 사용 <br />
✓ IronSessionApi 로그인 구현 <br />
✓ Cloutflare Images 사용 <br />
✓ Twilio 사용 <br />
✓ Sendgrid 사용 <br />
✓ middleware 사용 <br />
✓ vercel 배포 (carrot-market-bice.vercel.app)
